# يعني شنو AI Agents؟

الـ AI agent هو أداة يتديها هدف، تقرأ ملفات أو context، تقترح خطوات، وأحياناً تنفذ أفعال بالنيابة عنك. 

## تخيلها ببساطة

اعتبر الـ agent مساعد قاعد جنبك في التيرمنال. يقدر يقرأ المشروع، يقترح تعديلات، يشغل checks، ويواصل معاك من مهمة للتانية.

الكلمة المهمة هنا: بحذر. القرار النهائي عندك. راجع commands، افهم الخطوات الخطرة، واحمي معلوماتك الخاصة.

## ممكن يساعدوك في شنو؟

- بناء website أو portfolio.
- شرح code ما واضح.
- Debugging للأخطاء.
- كتابة scripts لحاجات مكررة في الكمبيوتر.

## الأداة الأساسية في الورشة

حنستخدم OpenCode كـ hands-on agent لأنه مناسب لشغل التيرمنال وبيدينا أداة واحدة نركز عليها كويس.

الأدوات التانية حتظهر في الـ catalog كمراجع. الهدف إنك تقارن وتفهم، ما إنك تثبت كل حاجة في سيشن واحد.

## القاعدة الذهبية

استخدم الـ agents عشان تسرع، ما عشان توقف تفكير. لو الـ agent حيعدل ملفاتك، يثبت software، يمسح حاجة، أو يطلب token، أبطأ وراجع الخطوة.
