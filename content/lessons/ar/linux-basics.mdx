# أساسيات Linux

Linux بظهر كتير في شغل AI حتى لو لابتوبك Windows أو macOS. سيرفرات الاستضافة، containers، وأمثلة التيرمنال غالباً بتستخدم أوامر قريبة من Linux.

## الخريطة في راسك

اتذكر ثلاث أفكار:

- الـ folder ممكن يتسمى directory.
- التيرمنال دايماً “واقف” داخل directory حالي.
- الأوامر غالباً بتأثر على الـ directory الحالي إلا لو كتبت path تاني.

## أوامر آمنة تعرفها

اطبع مكانك الحالي:

```powershell
pwd
```

النتيجة المتوقعة: يظهر path لمجلد.

اعرض الملفات:

```powershell
ls
```

النتيجة المتوقعة: تظهر الملفات والمجلدات في المكان الحالي.

ادخل مجلد:

```powershell
cd my-project
```

النتيجة المتوقعة: الـ prompt يتغير ويوريك إنك داخل المجلد الجديد.

## بطئ مع الأوامر الخطرة

انتبه للأوامر البتمسح، تكتب فوق ملفات، أو تعمل force. لو AI agent اقترح `rm` أو `del` أو `rmdir` أو `git reset` أو أي حاجة فيها `--force`، وقف واسأل عن شرح الأول.
