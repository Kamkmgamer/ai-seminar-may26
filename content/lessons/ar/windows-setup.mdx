# تجهيز ويندوز

الدرس ده بجهز ويندوز للشغل العملي مع AI. الهدف ما إنك تبقى خبير ويندوز. الهدف إن اللابتوب يكون جاهز للتيرمنال، التثبيت، المشاريع، و AI agents.

## محتاجين شنو؟

- متصفح حديث.
- Windows Terminal أو PowerShell.
- صلاحية تثبيت برامج.
- فولدر نحفظ فيه مشاريع الورشة.
- شوية صبر لو التيرمنال طلب منك تقفله وتفتحه تاني.

## افتح التيرمنال

اضغط زر Windows وابحث عن `PowerShell` أو `Terminal`.

اعمل الأمر ده:

```powershell
pwd
```

المفروض تشوف مكانك الحالي. `pwd` معناها "print working directory"، يعني التيرمنال واقف وين هسع.

## اعمل فولدر للورشة

```powershell
mkdir ai-workshop
cd ai-workshop
```

الحصل شنو: `mkdir` عمل فولدر، و `cd` دخل التيرمنال جوه الفولدر.

## ثلاث عادات في التيرمنال

في البداية محتاج بس الحاجات دي:

- `pwd` يوريك انت وين.
- `ls` يوريك الملفات في المكان الحالي.
- `cd folder-name` يدخلك فولدر.

جرب:

```powershell
pwd
ls
```

## نقطة أمان

ما تلصق أمر ما فاهمه، خصوصاً لو فيه كلمات زي `delete` أو `remove` أو `rm` أو `format` أو `token` أو `secret`.

لو AI agent اداك أمر، اسأله: "اشرح الأمر ده قبل ما أشغله."
