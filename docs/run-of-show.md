# Instructor Run Of Show

## Before Students Arrive

- Open `/en` and `/ar` once to confirm both locales load.
- Confirm the projector can show terminal text clearly.
- Keep the official references page open for changed install commands.
- Remind students not to paste secrets, tokens, `.env` files, or private links anywhere public.

## 0-15 Minutes: Orientation

- Start at the welcome lesson.
- Explain the workshop outcome: a safe deployed portfolio and a basic agent workflow.
- Show the language switcher, theme toggle, references page, and agent catalog.

## 15-40 Minutes: Agent Mental Model

- Use the agents lesson to separate chatbots from agents.
- Show the agent catalog as a comparison tool, not a ranking.
- Repeat the rule: the agent proposes, the student decides.

## 40-75 Minutes: Local Setup

- Walk through Windows setup where needed.
- Install Node and npm from official docs.
- Pause after each command for expected output checks.

## 75-115 Minutes: OpenCode And Portfolio

- Install OpenCode using current official docs.
- Open a small project and ask the agent for one small change first.
- Review the diff before accepting changes.
- Build the portfolio page in small, reversible steps.

## 115-145 Minutes: GitHub Safety And Deployment

- Create or open GitHub accounts.
- Check `.gitignore` before committing.
- Deploy with one provider guide, usually Vercel for Next.js.
- Submit only public project URLs if saved progress is enabled.

## Final 10 Minutes

- Mark completed lessons.
- Show the leaderboard only if nicknames and safe achievements are enabled.
- Point students to references and advanced sections for later study.
